# prd-design-implementation-agent

PRD・DesignDocを入力として受け取り、対象リポジトリへの実装変更プランを自動生成するエージェントです。

## 概要

このリポジトリは、プロダクト要求仕様書（PRD）と設計ドキュメント（DesignDoc）を入力として、対象リポジトリに対する最小限かつ的確な実装変更を自動的に計画・提案するエージェントの実装です。
